Two Pointers Pattern:

As the name suggests, there are 2 pointers to navigate through the array.

When to Use:

1. Used in Navigating Arrays or LinkedLists.
2. One Pointer can be at the beginning of the array and the other at the end of the array.
3. Two pointers can start at the same location but traverses at different speeds(Fast and Slow Pointers).
4. Works well with Sorted Arrays or strings. or that can be sorted.
5. Used when you need to find a pair, triplets that meet a condition.
6. Need to check if a string is Palindrome.
7. Need to find elenments that sum to target
8. Removing duplicates from sorted array.
